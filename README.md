# Crypto Bets

# Install dependencies

```
npm install
```

# Run project in dev mode (hot reloading)

```
npm run start
```

# Run ganache

```
ganache-cli
```

# Run truffle compile && migrate
````
truffle compile && truffle migrate
```

## Project should be available on localhost:1234

# Build project (For release)

```
npm run build
```
